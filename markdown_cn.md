# CSV工具箱 Pro

## 📋 概述

CSV工具箱 Pro 是一款功能全面的CSV文件处理工具，专为数据分析和数据处理人员设计。采用现代化的PySide6界面，支持高性能的大文件处理，集成了三大核心功能：文件合并、文件分割和数据提取。

## ✨ 主要特性

### 📁 文件合并
- 将多个CSV文件合并成一个文件
- 智能表头处理
- 支持大文件处理

### ✂️ 文件分割
- 三种分割方式：
  1. 按行数分割
  2. 按文件数分割
  3. 按大小分割
- 使用内存映射技术，快速处理大文件

### 🔍 数据提取
- 基于条件提取数据行
- 使用duckdb数据库引擎
- 自动编码检测和转换

# 运行程序
