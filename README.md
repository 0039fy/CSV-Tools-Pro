# CSV工具箱 Pro v3.2 使用说明

## 一、工具简介

专为CSV文件打造的高性能桌面工具，核心优势的是 **基于内存映射+多线程技术，极速处理GB级大文件**，无内存溢出，适配Windows全系列64位系统，核心支持合并、分割、数据提取三大功能。

### 核心特性

- 📁 大文件合并：智能处理表头，批量快速合并，适配各类编码

- ✂️ 灵活分割：按行数/文件数/大小分割，保留表头，保证行完整性

- 🔍 精准提取：大文件快速检索，列值筛选，支持空值/近似匹配

- ⚡ 极致性能：大文件分块读写，UI与处理分离，界面无卡顿

- 📑 编码自适应：自动检测/转换UTF-8/GBK/GB18030等编码，兼容Excel

- 📊 可视化：实时进度+彩色日志，操作结果一键查看

## 二、快速运行

无需安装，直接双击「CSV.Tools_Pro.exe」（Windows安全提示选**允许运行**），**安全**、**无毒**、**无后门**，请放心使用；首次使用数据提取功能需联网，自动加载duckdb依赖，启动后直接进入功能主界面。



## 三、核心功能操作

### 3.1 📁 文件合并

1. 点击「+添加」选择多份CSV文件，支持批量移除/清空列表

2. 选择输出路径，勾选「保留表头」（仅首文件保留，适配大文件批量合并）

3. 点击「🚀 开始合并」，完成后可一键打开输出目录，查看处理耗时/文件大小

4. **10GB**文件秒级合并，测试合并103个100M耗时**7.46**秒




### 3.2 ✂️ 文件分割

1. 浏览选择待分割大文件，按需任选一种分割方式：按行数/文件数/大小（默认100万行/10个/100MB）

2. 设置文件前缀（默认split_part）+输出目录，点击「✂️ 开始分割」

3. 分割后文件自动命名「前缀_0001.csv」，分割文件均保留原表头，GB级大文件极速分割

4. 测试**10GB**文件按行数（每个100万行）分割，耗时**28.37**秒；按文件数量（100个文件）分割耗时**26.83**秒；按文件大小（100M）分割耗时**21.14**秒





### 3.3 🔍 数据提取

1. 选择目标大文件，程序自动检测编码并转换，加载行列信息

2. 选择筛选列，可手动输入值/点击「获取示例值」快速选择，支持空值提取

3. 选择输出路径，点击「🔍 开始提取」，完成后查看匹配行数/执行耗时，结果为UTF-8-BOM编码，兼容Excel

4. 秒级提取特定数据，测试从10GB（5000万行+）大文件中提取指定列（SubNetwork）的某类型（3子网）14万行+数据，耗时**9.13**秒




## 四、大文件处理专属优化

- 内存映射：不加载整文件到内存，GB级文件轻松处理

- 分块读写：大文件分块操作，降低内存占用，提升处理速度

- 编码兼容：非UTF-8大文件自动转UTF-8临时文件处理，避免乱码

- 后台处理：大文件操作时UI无响应为正常现象，处理完成后自动更新进度/结果

## 五、常见问题

1. **文件处理无响应**：后台高速处理中，无需操作，完成后界面自动更新

2. **合并/分割后乱码**：用记事本打开文件→另存为→编码选「UTF-8」即可

3. **分割行数与设置不符**：自动保留表头，实际数据行=设置行数-1

4. **提取无结果**：程序自动去除列值首尾空格/换行，大文件示例值为随机采样，可重新获取
