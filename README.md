📖 使用指南

📁 文件合并
📥 添加文件：点击「+ 添加」按钮选择要合并的 CSV 文件（可多选）
📝 设置路径：设置输出路径（工具会自动生成默认路径，也可手动选择）
📋 表头设置：勾选「保留表头」选项（建议勾选，只保留第一个文件的表头）
🚀 开始合并：点击「🚀 开始合并」按钮，等待处理完成
📂 查看结果：可通过「打开输出目录」按钮直接查看生成的文件

✂️ 文件分割
📂 选择文件：点击「浏览」按钮选择要分割的 CSV 文件
⚙️ 选择方式：从下拉菜单选择分割方式（📏 按行数 / 📦 按文件数 / 📊 按大小）
🔢 设置参数：根据选择的分割方式设置对应的参数值，填写文件名前缀，选择输出目录
✂️ 开始分割：点击「✂️ 开始分割」按钮，等待处理完成
📊 查看日志：查看详细的分割结果日志，包含生成文件数量、行数、大小等信息

🔍 数据提取
📁 选择文件：点击「浏览」按钮选择要提取数据的 CSV 文件（工具会自动检测并转换编码）
📋 选择列名：从下拉列表选择要筛选的列（自动加载文件中的所有列）
🔤 输入值：
✍️ 手动输入要筛选的值
📋 或点击「获取示例值」按钮从列的示例值中选择
🧹 可点击「清空」按钮清空输入框和示例值列表
📤 输出设置：设置输出路径（工具会自动生成默认路径）
🔍 开始提取：点击「🔍 开始提取」按钮，查看匹配结果和详细日志

⚠️ 注意事项
大文件处理：
处理 GB 级文件时，请确保有足够的硬盘空间
分割大文件建议使用「按大小分割」模式
提取数据时，工具会自动转换编码为 UTF-8
编码问题：
工具会自动检测文件编码，无需手动设置
如遇到乱码，可尝试用记事本打开文件另存为 UTF-8 格式
性能优化：
处理超大文件时，建议关闭其他占用内存的程序
提取功能需要安装 duckdb，如未安装会自动提示
文件格式：
确保 CSV 文件格式规范，列分隔符为逗号
建议文件编码为 UTF-8，避免乱码问题
